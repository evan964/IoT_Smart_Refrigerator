# IoT_Smart_Refrigerator

## Poject description

Build a IoT system based on esp32 chip to get information in real time like temperature .

See all the features in the report document.

cloud : Adafruit

## Components
ESP32,
BME280,
TMP36,
Led,
Button,
Photoresistor,
HC-SC04 ultrasound sensor,
Servomotor
